## Features

- **Type Safety**: Automatically matches your Postgres types with Dart's type system.
- **Support for Multi-Environments**: Dedicated packages for core Dart, Flutter, and Serverpod.
- **Easy Initialization**: Interactive CLI tools for seamless project setup.
