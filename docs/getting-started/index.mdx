## Quick Start

### 1. Install

Add the appropriate package to your `pubspec.yaml`:

<Tabs items={["Flutter", "Pure Dart", "Serverpod"]}>
  <Tab id="flutter">
    ```yaml dependencies: supabase_codegen_flutter: ^4.0.0 ```
  </Tab>
  <Tab id="pure-dart">```yaml dependencies: supabase_codegen: ^4.0.0 ```</Tab>
  <Tab id="serverpod">
    ```yaml dependencies: supabase_codegen_serverpod: ^4.0.0 ```
  </Tab>
</Tabs>

### 2. Initialize

Run the initialization command to create a `.supabase_codegen.yaml` file:

<Tabs items={["Flutter", "Pure Dart", "Serverpod"]}>
  <Tab id="flutter">```bash dart run supabase_codegen_flutter:init ```</Tab>
  <Tab id="pure-dart">```bash dart run supabase_codegen:init ```</Tab>
  <Tab id="serverpod">```bash dart run supabase_codegen_serverpod:init ```</Tab>
</Tabs>

### 3. Generate

Generate your types with a single command:

<Tabs items={["Flutter", "Pure Dart", "Serverpod"]}>
  <Tab id="flutter">
    ```bash dart run supabase_codegen_flutter:generate_types ```
  </Tab>
  <Tab id="pure-dart">```bash dart run supabase_codegen:generate_types ```</Tab>
  <Tab id="serverpod">
    ```bash dart run supabase_codegen_serverpod:generate_types ```
  </Tab>
</Tabs>
